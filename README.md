# 📈 Dividenden-Tracker

Eine persönliche Web-App zur Dividenden-Planung mit Reinvestment-Simulation.

## Features

- Portfolio-Verwaltung mit Live-Kursen (yfinance)
- Sparraten pro Position (wöchentlich/zweiwöchentlich/monatlich)
- Dividenden-Kalender für die nächsten 12 Monate
- Langzeit-Simulation mit Reinvestment und deutschen Steuern
- Berücksichtigung von Sparerpauschbetrag und US-Quellensteuer

## Disclaimer

Diese App ist keine Anlage- oder Steuerberatung. Alle Simulationen basieren auf aktuellen Marktdaten und sind reine Projektionen.

## Tech-Stack

Python, Streamlit, yfinance, pandas, Plotly
